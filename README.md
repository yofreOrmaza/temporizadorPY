## Requirements

### Windows

- Install Python from Microsoft Store
- Install pygame from cmd: ```pip install pygame```

### Linux (Ubuntu)

- pip: ```sudo apt install python3-pip```
- pygame: ```sudo apt install python3-pygame```
- Tkinter: ```sudo apt install python3-tk```

### Linux (ArchLInux)

- pip: `sudo pacman -S python-pip`
- pygame: `sudo pacman -S python-pygame`
- Tkinter: `sudo pacman -S tk`

## How to Use

Add any mp3 file named “finish” to serve as an end-of-timer alert (finish.mp3)

### Run the program

To run the program, enter the command ```python3 main.py``` in CMD or Terminal.
